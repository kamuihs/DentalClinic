# Sistema de Prontuário Eletrônico para Clínicas e Consultórios

## Objetivo

Desenvolver uma plataforma SaaS para gestão completa de clínicas odontológicas, médicas e estéticas, contemplando prontuário eletrônico, agenda, financeiro, CRM, teleconsulta, assinatura digital, gestão de pacientes e automações com inteligência artificial.
